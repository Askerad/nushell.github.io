Expand a path to its absolute form

# Usage

> path expand ...args {flags}

# Parameters

  • ...args: Optionally operate by column path

# Flags

  • -h, --help: Display this help message

# Examples

Expand relative directories

> echo '/home/joe/foo/../bar' | path expand
