# min for expression

Creates a min expression

# Signature

> min

# Examples

Min aggregation for a group by

> [[a b]; [one 2] [one 4] [two 1]]
    | into df
    | group-by a
    | agg (col b | min)

# min for lazyframe

Aggregates columns to their min value

# Signature

> min

# Examples

Min value from columns in a dataframe

> [[a b]; [6 2] [1 4] [4 1]] | into df | min
