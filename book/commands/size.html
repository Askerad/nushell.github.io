# size

Gather word count statistics on the text.

# Signature

> size

# Examples

Count the number of words in a string

> "There are seven words in this sentence" | size

Counts unicode characters

> '今天天气真好' | size

Counts Unicode characters correctly in a string

> "Amélie Amelie" | size
