# skip while

Skip elements of the input while a predicate is true.

# Signature

> skip while (predicate)

# Parameters

  • predicate: the predicate that skipped element must match

# Examples

Skip while the element is negative

> echo [-2 0 2 -1] | skip while $it < 0
