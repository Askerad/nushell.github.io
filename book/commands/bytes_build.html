# bytes build

Create bytes from the arguments.

# Signature

> bytes build ...rest

# Parameters

  • ...rest: list of bytes

# Examples

Builds binary data from 0x[01 02], 0x[03], 0x[04]

> bytes build 0x[01 02] 0x[03] 0x[04]
